# Address Book Project (C Language)

##  Project Description
The Address Book project is a menu-driven application developed in **C language** that allows users to store and manage contact information efficiently.  
It demonstrates the use of **structures, file handling, pointers, and string operations**.

This project helps in understanding how real-world data can be stored permanently using files and accessed when required.

---

## Features
- Add new contacts
- View all saved contacts
- Search contacts by name or number
- Edit existing contact details
- Delete contacts
- Store data permanently using file handling

---

##  Technologies Used
- C Programming Language
- File Handling (`fopen`, `fread`, `fwrite`)
- Structures
- Pointers
- String handling functions
